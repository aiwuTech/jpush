jpush
=====

jpush api for golang

概述
----------------------------------- 
   这是JPush REST API 的 go 版本封装开发包,仅支持最新的REST API v3功能。
   REST API 文档：http://docs.jpush.cn/display/dev/Push-API-v3
   对所有官方支持的API都进行了封装

使用  
----------------------------------- 
   
   `go get https://github.com/isdamir/jpush`
   
   然后请参照example/example.go使用
